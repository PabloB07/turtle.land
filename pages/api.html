API / Node

Turtle.Land Node

You can use Turtle.Land api as public node for SimpleWallet
Node IP is api.turtle.land on port 80
You can run SimpleWallet as simplewallet.exe --remote-daemon api.turtle.land
Best option for SimpleWallet remote note is public node on TurtleNode.io
Choose node what is closer to your location.


Supported APIs

  • hashrate – current estimated network hashrate
  • height – current height (incl. genesis block)
  • reward – current block reward
  • supply – total available supply
